혼자 공부하는 자바

프로그래밍 언어를 처음 배우는 사람에게 추천하는 교재입니다.
사실상 입문 파트만 다루고, 매우 단순하며, 시각 자료가 매우 풍부합니다.
단점으로는, Collection, Threads 등 조금 난이도가 있는 부분은 상세히 다루지 않았기 때문에,
이 책을 다 본 뒤에 추가적인 학습이 요구됩니다.

생활코딩! 자바 프로그래밍 입문

페이스북에서 유명한 코딩 그룹인 '생활 코딩'을 운영하시는 이고잉님이 집필한 교재입니다.
입문자의 눈높이에 맞게 작성되었기 때문에 프로그래밍 언어를 처음 배우는 사람도 이해할 수 있습니다.
다만, 저자분이 밝혔듯이 타 자바 교재 저자에 비해 자바에 대한 숙련도가 낮기 때문에,
전문성은 조금 떨어질 수 있습니다.

Do it! 자바 프로그래밍 입문

C 언어나 Python 등 다른 프로그래밍 언어를 학습했던 사람에게 추천하는 교재입니다.
유튜브 에 저자가 직접 강의한 강의영상이 무료로 남아있으며,
동영상 강의와 함께 보면 더욱 빠르게 자바 언어를 학습할 수 있습니다.

Java의 정석

대한민국 Java 교재 중 가장 유명한 자바의 정석입니다.
C 언어나 Python 등 다른 프로그래밍 언어를 학습했던 사람에게 추천하는 교재입니다.
교재가 두꺼운 만큼 풍부한 내용과 코드 예제가 담겨 있으며,
유튜브 에서 저자가 직접 강의한 강의영상을 시청할 수 있습니다.
